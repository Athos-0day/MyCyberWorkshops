# Défi 1

L'objectif ici est de mettre en valeur le risque lorsqu'on réutilise une clé.

Pour cela vous trouverez les fichiers **cipher1.bin** et **cipher2.bin** qui contiennent les binaires des messages 1 et 2. Ces deux messages ont été chiffrés en XOR avec la même clé de chiffrement. 

N'hésitez pas à demander des indices pour avancer, les deux messages sont des clubs de l'n7 et possèdent une majuscule en début de message.


Pour lire les fichiers il est conseillé d'utiliser le mode **rb**.
